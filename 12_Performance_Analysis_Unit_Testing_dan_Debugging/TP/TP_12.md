## Soal

## Kode Program

## Deskripsi

Program ini bertujuan untuk memproses berbagai jenis data yang tersimpan dalam sebuah array. Setiap data akan diperiksa apakah merupakan representasi angka atau teks. Jika data berupa angka, maka nilainya akan dikalikan dua. Jika bukan angka, maka program akan menampilkan teks beserta panjang karakter yang dimilikinya.

Namun, pada kode yang diberikan terdapat sebuah defect yang menyebabkan program berhenti ketika memproses data yang bukan bertipe string. Oleh karena itu dilakukan analisis terhadap penyebab error, kemudian dilakukan perbaikan agar seluruh data dapat diproses dengan benar.

---

## Analisis Defect

Defect utama terdapat pada fungsi `processData()`, yaitu pada baris berikut:

```javascript
const str = data.toLowerCase();
```

Method `toLowerCase()` hanya dimiliki oleh objek bertipe **String**. Sementara itu, array `data` berisi berbagai tipe data seperti `Number` dan `Boolean`.

Ketika program mencoba menjalankan:

```javascript
456.toLowerCase();
```

atau

```javascript
true.toLowerCase();
```

program akan menghasilkan error:

```text
TypeError: data.toLowerCase is not a function
```

Akibatnya program berhenti sebelum seluruh data selesai diproses.

---

## Perbaikan Program

Perbaikan dilakukan dengan mengubah seluruh data menjadi string terlebih dahulu menggunakan fungsi `String()` sebelum memanggil `toLowerCase()`.

Perubahan kode:

Sebelum:

```javascript
const str = data.toLowerCase();
```

Sesudah:

```javascript
const str = String(data).toLowerCase();
```

Dengan perubahan tersebut, semua tipe data dapat diproses sebagai string sehingga program dapat berjalan tanpa menghasilkan error.

---

## Alur Program

1. Program membuat array yang berisi beberapa tipe data.
2. Setiap elemen array diproses menggunakan fungsi `processData()`.
3. Data terlebih dahulu dikonversi menjadi string.
4. Program mencoba mengubah string menjadi bilangan menggunakan `parseInt()`.
5. Jika data merupakan bilangan bulat, hasilnya dikalikan dua.
6. Jika bukan bilangan bulat, program menampilkan teks beserta jumlah karakternya.
7. Hasil setiap proses ditampilkan ke layar.

## Penjelasan

1. Data `"123"` dikenali sebagai angka sehingga hasilnya menjadi `246`.
2. Data `456` berhasil diproses setelah dikonversi menjadi string, sehingga menghasilkan `912`.
3. Data `"hello"` dianggap sebagai teks dan panjangnya dihitung sebanyak `5` karakter.
4. Data `78.9` tidak memenuhi syarat sebagai bilangan bulat sehingga diperlakukan sebagai teks.
5. Data `true` juga diperlakukan sebagai teks dengan panjang `4` karakter.

---

## Kesimpulan

1. Defect pada program disebabkan oleh penggunaan method `toLowerCase()` pada data yang bukan bertipe string.
2. Error tersebut menyebabkan program berhenti ketika memproses data bertipe `Number` atau `Boolean`.
3. Perbaikan dilakukan dengan mengonversi data menjadi string menggunakan `String(data)` sebelum memanggil `toLowerCase()`.
4. Setelah diperbaiki, seluruh elemen array dapat diproses dengan benar tanpa menghasilkan error.
