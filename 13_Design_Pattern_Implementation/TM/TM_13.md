## Deskripsi

Event delegation merupakan sebuah teknik dalam JavaScript yang memanfaatkan mekanisme **event bubbling**, yaitu proses ketika suatu event yang terjadi pada elemen anak akan diteruskan ke elemen induknya. Dengan memanfaatkan mekanisme tersebut, event listener tidak perlu dipasang pada setiap elemen secara terpisah, melainkan cukup dipasang pada elemen parent.

Teknik ini sering digunakan ketika terdapat banyak elemen yang memiliki perilaku sama atau ketika elemen-elemen tersebut dibuat secara dinamis menggunakan JavaScript. Dengan demikian, pengelolaan event menjadi lebih sederhana dan efisien.

---

## Cara Kerja Event Delegation

1. Sebuah event listener dipasang pada elemen parent.
2. Ketika elemen anak memicu suatu event (misalnya klik), event tersebut akan naik menuju parent melalui proses **event bubbling**.
3. Parent menerima event tersebut dan memeriksa elemen yang memicunya menggunakan `event.target`.
4. Program menjalankan aksi yang sesuai berdasarkan elemen yang dipilih.

---

## Contoh Implementasi

```javascript
const daftar = document.getElementById("daftar");

daftar.addEventListener("click", function(event) {
    if (event.target.tagName === "BUTTON") {
        console.log("Tombol diklik:", event.target.textContent);
    }
});
```

Pada contoh di atas, event listener hanya dipasang pada elemen `daftar` sebagai parent. Ketika salah satu tombol di dalamnya diklik, parent akan menerima event tersebut dan mengetahui tombol mana yang ditekan melalui `event.target`.

---

## Keunggulan Event Delegation

1. Mengurangi jumlah event listener yang digunakan sehingga penggunaan memori lebih efisien.
2. Mempermudah pengelolaan banyak elemen yang memiliki fungsi serupa.
3. Mendukung elemen yang ditambahkan secara dinamis tanpa perlu menambahkan event listener baru.
4. Membuat kode menjadi lebih ringkas dan mudah dipelihara.

---

## Kekurangan Event Delegation

1. Hanya dapat digunakan pada event yang mendukung proses **event bubbling**.
2. Membutuhkan pengecekan terhadap `event.target` untuk mengetahui elemen yang memicu event.
3. Pada struktur DOM yang sangat kompleks, proses identifikasi target dapat menjadi lebih rumit.

---

## Penjelasan

Menurut pemahaman saya, event delegation adalah teknik yang memanfaatkan event bubbling untuk menangani banyak event menggunakan satu event listener pada elemen parent. Dengan cara ini, kode menjadi lebih efisien karena tidak perlu membuat event listener untuk setiap elemen secara terpisah.

Teknik ini sangat berguna pada aplikasi web modern yang memiliki banyak komponen interaktif atau elemen yang ditambahkan secara dinamis, seperti daftar tugas, tabel data, menu, atau daftar komentar.

## Kesimpulan

1. Event delegation merupakan teknik penanganan event dengan memanfaatkan mekanisme event bubbling.
2. Event listener cukup dipasang pada elemen parent sehingga dapat menangani event dari seluruh elemen anak.
3. Teknik ini membuat penggunaan memori lebih efisien dan kode lebih mudah dikelola.
4. Event delegation sangat cocok digunakan pada aplikasi yang memiliki banyak elemen atau elemen yang berubah secara dinamis.
