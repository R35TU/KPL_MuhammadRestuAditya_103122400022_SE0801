## Soal

Buatlah sebuah pustaka JavaScript bernama **mtk-gampang** dengan struktur sebagai berikut:

```text
|   index.js
|   package.json
\---lib
        pangkat.js
        bulat.js
        kuadrat.js
```

Setiap berkas pada folder `lib` hanya memiliki satu fungsi:

1. `pangkat(x, y)` untuk menghitung nilai x pangkat y.
2. `bulat(x)` untuk mengubah bilangan desimal menjadi bilangan bulat (contoh: `-4.25` menjadi `-4`).
3. `kuadrat(x)` untuk menghitung akar kuadrat dari suatu bilangan.

Fungsi-fungsi tersebut harus diakses melalui `index.js` sebagai entry point library.

## Deskripsi

Program ini merupakan sebuah library matematika sederhana yang menyediakan tiga fungsi utama, yaitu menghitung pangkat, membulatkan bilangan desimal menjadi bilangan bulat tanpa pecahan, dan menghitung akar kuadrat.

Masing-masing fungsi ditempatkan pada file yang berbeda di dalam folder `lib` sehingga lebih terstruktur dan mudah dipelihara. Seluruh fungsi kemudian diekspor kembali melalui `index.js`, sehingga pengguna library cukup melakukan import dari satu file utama tanpa mengakses file di dalam folder `lib` secara langsung.

---

## Alur Program

1. Pengguna mengimpor fungsi dari `index.js`.
2. `index.js` mengambil fungsi dari masing-masing file pada folder `lib`.
3. Fungsi `pangkat(x, y)` menghitung nilai x pangkat y menggunakan operator eksponensial (`**`).
4. Fungsi `bulat(x)` mengubah bilangan desimal menjadi bilangan bulat menggunakan `Math.trunc()`.
5. Fungsi `kuadrat(x)` menghitung akar kuadrat menggunakan `Math.sqrt()`.
6. Nilai hasil perhitungan dikembalikan dan dapat digunakan pada program utama.

---

## Output

Program menghasilkan output sebagai berikut:

```
Seorang insinyur menetapkan luas panel 3 meter persegi, lalu menggunakan kapasitas penyimpanan sebesar 1024 watt-jam. Ketika sensor mengirimkan data arus sisa yang berantakan seperti 85.95 ampere, ia kalibrasikan menjadi 85 agar sistem keamanan memutus aliran tepat pada angka bulat tanpa koma.
```

---

## Penjelasan

1. `kuadrat(12)` menghasilkan nilai sekitar `3.464`, kemudian `bulat()` mengubahnya menjadi `3`.
2. `pangkat(2, 10)` menghasilkan nilai `1024`.
3. `bulat(85.95)` menghasilkan nilai `85` dengan menghilangkan bagian desimal.

---

## Kesimpulan

1. Library berhasil menyediakan tiga fungsi matematika sederhana yang dapat digunakan kembali pada program lain.
2. Setiap fungsi dipisahkan ke dalam file yang berbeda sehingga struktur proyek menjadi lebih rapi.
3. Seluruh fungsi diekspor melalui `index.js` sebagai entry point library sesuai ketentuan soal.
4. Penggunaan modularisasi memudahkan pengembangan dan pemeliharaan kode di masa mendatang.
